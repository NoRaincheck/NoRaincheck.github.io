## Testing Go with Monty (Python)

_April 2026_

Not much to say right now besides it works. 

```go
package main

import (
	"context"
	"fmt"
	"log"

	monty "github.com/ewhauser/gomonty"
)

var a = `
def add(x):
    return x + 1

add(x)
`

func main() {
	runner, err := monty.New(a, monty.CompileOptions{
		Inputs: []string{"x"},
	})
	if err != nil {
		log.Fatal(err)
	}

	value, err := runner.Run(context.Background(), monty.RunOptions{Inputs: map[string]monty.Value{"x": monty.Int(10000)}})
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(value.Raw())
}
```
